Host

The host parameter sets hosts on the instances. Setting this, every node will have the host.

Usage

   1  has_host(:name => '...') do
   2  	# More options. 
   3  	# This block is optional
   4  end

Options

name
The name of the instance
ip
IP address of the instance for the host entry

Examples

   1  has_host({:name => "other_machine", :ip => 192.168.0.101 })
