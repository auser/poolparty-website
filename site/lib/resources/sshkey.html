Ssh Key

The sshkey resource specifies an ssh key that should be distributed on all the nodes

Usage

   1  has_sshkey(:key => '...') do
   2  	# More options. 
   3  	# This block is optional
   4  end

Options

key
The key content for the ssh key
target
The location of the ssh key

Examples

   1  has_sshkey(:key => "ABIGLONGSTRINGOFDIGETS", :target => "/root/.ssh/key_file")
