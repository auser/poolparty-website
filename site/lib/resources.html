PoolParty resources

This is a list of resources currently available at your disposal when writing your spec-file and plugins. All of these resources can be called with the functions:

  • has_
    • For example has_file(...)
  • does_not_have_
    • For example: does_not_have_file(...)

Resources

  • file
  • package
  • gem
  • exec
  • variable (for templates)
  • line_in_file
  • remotefile
  • cron
  • host
  • service
  • sshkey
  • symlink
  • directory
  • classpackage

Helper methods
