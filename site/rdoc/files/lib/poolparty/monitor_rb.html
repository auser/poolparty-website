Monitor

This provides the interface to monitors in a clouds.rb configuration file

Usage

  PoolParty::Monitor.new(:cpu) do |c|
    vote_for(:expand) if c > 0.8
    configure if c < 0.1
  end

The methods inside the block will be executed on the cloud with the value retrieved by the monitor.

[Validate]
