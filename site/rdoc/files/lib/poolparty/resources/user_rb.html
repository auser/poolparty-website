User

The user resource ensures there is a user created on the nodes

Usage

  has_user(:name => '...') do
    # More options.
    # This block is optional
  end

Options

  • name Name of the service to be running
  • comment Comment about the user
  • uid uid of the user
  • gid gid of the user
  • home Home directory of the user
  • shell Shell type of the user
  • password The hashed password

Examples

  has_user "fred" do
    home "/home/fred"
  end

[Validate]
