Mount

The mount specifies a mount that is to be mounted on the instances

Usage

  has_mount(:name => '...') do
    # More options.
    # This block is optional
  end

Options

  • name The location of the mount (default: /data)
  • device The device location for the mount. This mounts at the directory set by the name
  • options The options to be set in the mount file fstab (default: rw,nosuid,noquota)
  • fstype The Type of mount (default: xfs)

Examples

  has_mount(:name => "/data", :device => "/dev/sda100")
