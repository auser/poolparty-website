Route

  Manage the system routing table.

Usage

  has_route "10.0.0.1" do
    gateway "10.0.0.20"
    metric 5
    route_type :net
    netmask "255.255.0.0"
  end

Options

  • gateway The gateway for the route
  • metric The route metric as an integer
  • route_type The type of route as a symbol, either :host or :net
  • netmask The decimal representation of the network mask eg 255.255.255.0
  • device Network interface to apply this route to

[Validate]
