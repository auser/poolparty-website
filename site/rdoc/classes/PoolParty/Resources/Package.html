Package

The package resources defines a package that must be present on all of the instances This will install the "name_of_package" package with the package provider (apt, yum, etc)

Usage

  has_package "name_of_package"
  has_package(:name => '...') do
    # More options.
    # This block is optional
  end

Options

  • name The package name. The default provider for your OS will be picked by the dependency resolver

Examples

  has_package(:name => 'apache2')
