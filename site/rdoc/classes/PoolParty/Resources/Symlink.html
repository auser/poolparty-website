Symlink

The symlink resource sets a symlink

Usage

has_symlink(:key => '...') do
  # More options.
  # This block is optional
end

Options

  • name The location of the symlink
  • target The source of the symlink, the existing file or directory

Examples

has_symlink(:name => "/var/www/poolpartyrb.com/public", :source => "/var/www/poolpartyrb.com/poolparty-website/site")
