Symlink

The symlink resource sets a symlink

Usage

   1  has_symlink(:key => '...') do
   2  	# More options. 
   3  	# This block is optional
   4  end

Options

Name
The location of the symlink
target
The source of the symlink, the existing file or directory

Examples

   1  has_symlink(:name => "/var/www/poolpartyrb.com/public", :source => "/var/www/poolpartyrb.com/poolparty-website/site")
