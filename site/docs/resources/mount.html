Mount

The mount specifies a mount that is to be mounted on the instances

Usage

   1  has_mount(:name => '...') do
   2  	# More options. 
   3  	# This block is optional
   4  end

Options

name
The location of the mount (default: /data)
device
The device location for the mount. This mounts at the directory set by the name
options
The options to be set in the mount file fstab (default: rw,nosuid,noquota)
fstype
The Type of mount (default: xfs)

Examples

   1  has_mount(:name => "/data", :device => "/dev/sda100")
