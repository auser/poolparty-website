Gempackage

Gempackages describe distributed gems on the cloud. The gem is only downloaded once and then hosted on the master if a download url is given.

Usage

   1  has_gempackage(:name => '...') do
   2  	# More options. 
   3  	# This block is optional
   4  end

Options

name
The gem name
download_url
The location of the gem to download and server across the instances
version
The gem version requirement (optional and useless if download_url is given)
source
The gem source (optional and useless matter if download_url is given)

Examples

   1  has_gempackage(:name => 'rake', 
   2  	:download_url => 'http://rubyforge.org/frs/download.php/43954/rake-0.8.3.gem')
