    Cloud

    All of the cloud binaries are labeled with

    -n [name of the cloud]

    • List
      List the current running and pending instances for the cloud
    • Start
      Start the cloud
    • Ssh
      Ssh into the cloud (specify the node with -i [defaults to the master 0])
    • Terminate
      Terminate the cloud
    • Refresh
      Reconfigure the cloud quickly. Useful if you want the cloud to refresh from the configuration
    • Provision
      Provision the cloud. When you make a change to your manifest, use this to push the changes to the cloud
    • Osxcopy
      Copy the master ip to the pasteboard (on osx)
    • Maintain
      Run the maintenance script to ensure the cloud is maintaining status
    • Expand
      Expands the cloud if it can expand
    • Contract
      Contract the cloud if it can contract
    • Configure
      Configure the cloud
    • Add-keypair
      Helper method to create a new keypair for the instances
