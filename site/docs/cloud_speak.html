    Cloud

    All of the cloud binaries have the following switches available 

    -n [name of the cloud]
-c [path/to/clouds.rb]
-h

    • List
      List the current running and pending instances for the cloud
    • Start
      Start the cloud
    • Ssh
      Ssh into the cloud (specify the node with -i [defaults to the master 0])
    • Terminate
      Terminate the cloud
    • Expand
      Expands the cloud if it can expand
    • Contract
      Contract the cloud if it can contract
    • Configure
      Configure the cloud
    • Console
      Give an irb console with the clouds.rb loaded
    • Reboot
      Restart each node in the cloud (one-by-one) to refresh the user-data and the nodes
    • Show
      Describe the cloud setup
